# 랜딩 페이지 전체 텍스트 · ChatGPT 분석용

이 문서는 메인 랜딩(`app/page.tsx` 및 관련 컴포넌트)에 **실제로 표시되는 카피·버튼·안내**를 위에서 아래 순서로 옮긴 것입니다.  
분석·카피 검수·컴플라이언스 검토 시 이 파일만 넘겨도 페이지 구조와 문구를 재구성할 수 있습니다.

**관련 소스:** `app/page.tsx`, `components/landing/*`, `app/layout.tsx`(메타), `lib/privacy-consent-full-text.ts`(개인정보 전문)

## 4. 자가 체크 문항 (`CheckQuestionsSection`, `id="check-questions"`)

**제목·부제**

- 뇌·심장 치료가 반복될 때  
- 내 보험도 대비되어 있을까요?  
- 아래 항목 중 해당되는 내용을 선택해주세요  
- 2개 이상 해당된다면  
- 내 보험 구조를  
  한 번 확인해보는 것이 좋습니다  

**선택 항목 (토글, 여러 개 선택 가능)**

1. 어지럼·두통을 자주 느낀다  
2. 가슴 답답함·두근거림이 있다  
3. 가족 중 뇌·심장 질환 병력이 있다  
4. 내 보험에  
   뇌·심장 치료비 보장이 있는지 모른다  
5. 진단금은 있지만 치료비 보장은 모른다  

**버튼:** 내 보험 보장 구조 확인하기  

**동작 요약:** 버튼 클릭 시 선택된 항목 개수를 집계하고, 결과 구역을 표시한 뒤 `result` 섹션으로 스크롤합니다.

## 7. 신청 폼 (`ApplicationFormSection`, `id="application-form"`)

**제목·설명**

- 전화 상담 강요나 보험 가입 권유 없이,  
- 현재 보장 구조만  
  객관적으로 확인해드립니다  

**입력 필드**

| 라벨 | placeholder / 형식 |
|------|-------------------|
| 이름 | 이름을 입력해주세요 |
| 전화번호 | 010-0000-0000 (자동 하이픈, 숫자만 저장) |

**동의**

- 체크박스 라벨: [필수] 개인정보 수집 및 이용에 동의합니다  
- 상세 열기 버튼: [내용 보기]  
- 상세 영역: `PRIVACY_CONSENT_FULL_TEXT` 전문 표시 (스크롤 가능 영역, 최대 높이 제한 있음)

**검증 메시지 (오류 시)**

- 개인정보 수집 및 이용에 동의해주세요.  
- 이름과 전화번호를 모두 입력해주세요.  
- 전화번호를 올바르게 입력해주세요.  
- (서버 오류 시) `submitLandingLead` 반환 메시지 표시  

**제출 버튼**

- 기본: 내 보험 구조 확인 요청하기  
- 전송 중: 요청 중...  
- 비활성: 미동의 또는 전송 중  

**하단 안내:** 입력하신 정보는 보장 확인 목적에만 사용됩니다

## 8. 결과 카드 (`ResultSection`, `id="result"` · 문항 제출 후에만 노출)

**분기 규칙:** 선택 항목 개수 **≥ 2** → “확인 필요” 버전. 그 외 → “확인 권장” 버전.

### 8-A. 선택 2개 이상 (`확인 필요`)

- 배지: 확인 필요  
- 제목: 보장 구조 확인이 필요한 상태입니다  
- 본문:  
  - 현재 보험이  
  - 진단금 1회 중심인지,  
  - 치료비 반복 보장 구조인지 확인이 필요합니다  
- 강조 박스:  
  - 지금 확인하지 않으면  
  - 실제 치료 상황에서 보장 차이를  
  - 뒤늦게 알게 될 수 있습니다  
- 버튼: 내 보험 구조 확인 요청하기 → `#application-form` 스크롤  

### 8-B. 선택 2개 미만 (`확인 권장`)

- 배지: 확인 권장  
- 제목: 현재 상태는 양호할 수 있습니다  
- 본문:  
  - 증상이 없더라도  
  - 보장 구조는 보험증권을 확인해야 알 수 있습니다  
- 버튼: 내 보험 구조 확인 요청하기 → `#application-form` 스크롤

**혜택 태그:** 1:1 확인 · 무료 확인 · 빠른 답변  

**추가 안내:** 가입 내역이 있다면 / 함께 보내주세요  

**QR 안내:** 아래 QR코드를 스캔하세요

**링크 버튼:** 카카오톡으로 간편하게 확인하기  

**링크 URL:** 환경변수 `NEXT_PUBLIC_KAKAO_CHAT_URL` 이 있으면 우선, 없으면 기본값 `https://open.kakao.com/o/scS4vMoi`

## 10. FAQ (`FAQSection`)

**섹션 제목:** 자주 묻는 질문  

**항목**

1. **Q:** 정말 무료인가요?  
   **A:** 네. 보장 구조 확인은 무료입니다.  

2. **Q:** 가입 권유를 받게 되나요?  
   **A:** 원하지 않으시면 진행하지 않습니다.  

3. **Q:** 무엇을 확인하나요?  
   **A:** 진단금 중심인지 치료비 구조인지 확인합니다.  

4. **Q:** 기존 보험을 해지해야 하나요?  
   **A:** 기존 보험 해지 여부와는 별개로,  
   현재 보장 구조를 먼저 확인하는 과정입니다.  

5. **Q:** 보험증권 없어도 되나요?  
   **A:** 가능합니다. 앱 화면으로도 확인 가능합니다.

## 13. 개인정보 동의 전문 (폼 `[내용 보기]`)

폼에서 펼쳐 보이는 긴 약관 전문은 코드 상 **`lib/privacy-consent-full-text.ts`** 의 `PRIVACY_CONSENT_FULL_TEXT` 한 덩어리와 동일합니다.  
원문 파일로 관리한다면 **`data/privacy-consent-full.txt`** 를 참고하세요(주석 기준).

ChatGPT에 **전문까지** 필요하면 해당 `.ts` 또는 `.txt` 파일을 함께 첨부하는 것이 정확합니다.
